# 🍽 메뉴 추천 앱

사용자의 상황과 취향을 기반으로 메뉴를 추천해주는 웹 애플리케이션입니다. 반복되는 식사 고민을 줄이기 위해 제작했고, 간단한 입력만으로 빠르게 메뉴를 추천받을 수 있도록 했습니다.

## 배포 링크
https://menu-project-1jdb.onrender.com/

## 주요 기능
- 상황 기반 추천 : 인원 수와 시간대(아침/점심/저녁)을 입력하면 입력 조건에 맞는 메뉴를 필터링하여 추천해줍니다.
- 태그 기반 추천 : 면, 국물, 고기 등 원하는 특성을 선택하면 선택한 태그 중에 하나라도 해당되는 메뉴를 추천해줍니다.

## 기술 스택
- Python / Flask
- HTML / CSS / JavaScript
- Render, Gunicorn

## 구현 방식
### 데이터 설계 및 로딩 구조
- 메뉴 데이터는 JSON 파일로 분리하여 관리했습니다. 각 메뉴는 이름(name), 제공 시간대(meal_time), 추천 인원 범위(min_people, max_people), 태그(tags) 정보를 포함하며, 조건 기반 필터링이 가능하도록 설계했고, 데이터와 로직을 분리하여 확장성을 고려했습니다.
### 추천 로직
- 상황 기반 추천은 입력된 시간대가 meal_time에 포함되는지, 인원 수가 min_people과 max_people 사이의 범위에 해당하는지를 검사한 뒤, 모든 조건을 만족하는 메뉴를 후보로 수집 후 무작위로 추천하도록 했습니다.
- 태그 기반 추천은 사용자가 선택한 태그 중 하나라도 포함된 메뉴를 찾는 OR 조건 방식으로 구현했으며, 그렇게 선택된 후보 목록 중 무작위로 하나를 추천하도록 했습니다.
###서버 및 화면 구성
- Flask 라우트를 기능 단위로 분리하여 구성하였습니다. HTML 템플릿을 사용하여 입력 페이지와 결과 페이지를 분리하고 서버에서 전달한 데이터를 기반으로 결과를 출력하도록 구현했습니다.
### 배포
- Gunicorn과 Render를 사용하여 실제 서비스 환경에 배포했습니다.
